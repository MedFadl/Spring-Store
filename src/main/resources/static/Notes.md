Spring anotations Annotations to remind myself.

| Annotation               | Purpose                         |
|--------------------------|---------------------------------|
| `@SpringBootApplication` | Main Spring Boot app annotation |
| `@Controller`            | Handles web requests/views      |
| `@Service`               | Business logic layer            |
| `@Repository`            | Database/data access layer      |
| `@Component`             | Generic Spring bean             |
| `@RequestMapping`        | Maps URLs to methods            |
| `@Value`                 | Injects configuration values    |
| `@AutoWired`             | Selects default constructor     |
